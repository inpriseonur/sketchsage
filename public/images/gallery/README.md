# Landing Page Gallery Images

Bu klasörde landing page'de gösterilecek örnek çalışma görselleri bulunmalıdır.

## Gerekli Görseller

1. `example1.jpg` - Karanlık portre çalışması
2. `example2.jpg` - Çiçek çalışması
3. `example3.jpg` - Mimari çizim
4. `example4.jpg` - Doğa manzarası

## Hero Görseli

- `hero-example.jpg` - Ana sayfa hero bölümündeki büyük örnek görsel (mavi sulu boya)

## Önerilen Boyutlar

- Gallery: 600x800px (3:4 aspect ratio)
- Hero: 1000x1000px (square)
- Format: JPG veya PNG
- Dosya boyutu: Max 500KB per image

## Nasıl Değiştirilir?

1. Yeni görselleri bu klasöre (`public/images/`) yükleyin
2. Dosya adlarını yukarıdaki isimlere göre ayarlayın
3. Veya admin panelden (ileride) değiştirebileceksiniz

## Şu Anki Durum

⚠️ Placeholder görseller henüz eklenmemiştir. Lütfen gerçek görselleri yükleyin veya tasarımdan alınan görselleri kullanın.
