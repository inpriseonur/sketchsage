# Auth Page Example Images

Bu klasörde auth sayfalarının sol tarafında gösterilecek örnek görseller bulunmalıdır.

## Gerekli Görseller

1. `charcoal.jpg` - Kara kalem çalışma örneği
2. `watercolor.jpg` - Sulu boya çalışma örneği
3. `lineart.jpg` - Çizgi sanatı örneği

## Önerilen Boyutlar

- En az: 800x600px
- Önerilen: 1200x900px
- Format: JPG veya PNG
- Dosya boyutu: Max 500KB

## Nasıl Değiştirilir?

1. Yeni görselleri bu klasöre yükleyin
2. Admin panelden **Auth Examples** bölümüne gidin
3. Her örnek için:
   - Görseli yükleyin
   - Başlık ve açıklama girin
   - Sıralamayı ayarlayın
   - Kaydedin

## Şu Anki Durum

⚠️ Placeholder görseller henüz eklenmemiştir. Lütfen gerçek görselleri yükleyin.
